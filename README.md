# Basic Queries

A collection of SQL exercises focused on writing foundational \`SELECT\` queries using the **Northwind** database — a classic sample database used for learning SQL.

## About This Project

This was one of my first SQL exercises at **Year Up United — Learn to Code Academy (LTCA)**. The goal was to get comfortable reading data from a relational database using the \`SELECT\` statement and basic clauses.

## What I Learned

- Writing \`SELECT\` statements to retrieve specific columns from a table
- Filtering results with \`WHERE\` clauses
- Sorting data with \`ORDER BY\` (ascending and descending)
- Chaining multiple conditions with \`AND\` / \`OR\`
- Understanding how relational databases store and organize data

## Tech Stack

- **SQL** (MySQL)
- **Northwind** sample database

## Topics Covered

| Topic | Description |
|-------|-------------|
| \`SELECT\` | Retrieving specific columns and all columns |
| \`WHERE\` | Filtering rows based on conditions |
| \`ORDER BY\` | Sorting query results |
| \`LIMIT\` | Restricting the number of returned rows |
| Comparison operators | \`=\`, \`<\`, \`>\`, \`>=\`, \`<=\` |
